# master-code

Code used and developed during my master thesis. This repo includes preprocessing and templatematching python notebooks, to analyze the sample of interest. GARM SPED data aqcuired my Emil Christiansen, and 2100F SPED data acquired by Ursula Ludacka.

Dependencies found in "pyproject.toml"

# Contents
- 01 SPED preprocessing notebooks for 2100F and GARM data
- 02 Templatematching notebook tested and developed for GARM data
- 03 roi based workflow, where regions of interest (ROIs) are defined and further template matched in 04
- 04 template matching rois
- 05 visualizing and validating the template matched rois based on criteria
